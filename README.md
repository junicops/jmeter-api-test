# JMeter API Test Project

Project ini digunakan untuk testing API menggunakan Apache JMeter.

## Purpose
- Test login API
- Test create & get booking
- Validate response using JSON Assertion
- Load testing sederhana

-------------------------------------------------

## Test Scenario
1. Login / Auth
2. Extract token (JSON Extractor)
3. Create booking
4. Get booking by ID
5. Assertion response

-------------------------------------------------

## 🛠 Tools
- Apache JMeter
- API: ReqRes / Restful Booker
- Git for version control

-------------------------------------------------

## 📁 Project Structure

tests/ → JMeter test plan (.jmx)
data/ → CSV test data
config/ → config files
results/ → JMeter result (.jtl)
reports/ → HTML reports
